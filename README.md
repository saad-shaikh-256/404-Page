# 404 Error Page - HTML & CSS

![404 Error Page](https://github.com/saad-shaikh-256/404-Page/blob/main/Assets/Images/Full%20Cover.jpg)

## Overview

This project is a **404 Error Page** built using **HTML** and **CSS**. It features a modern and responsive design that provides a simple and user-friendly "Page Not Found" experience.

## Features

- **Clean & Modern Design**: Minimalistic 404 page layout.
- **Responsive**: Works on all devices - Desktop, Tablet, and Mobile.
- **Customizable**: Easy to update text and design.

## Technologies Used

- **HTML**
- **CSS**
